RxList/RxMap/RxSet は axii の組み込みリアクティブデータ構造です。
これらの API は JavaScript の Array/Map/Set に似ています。
より詳細な例については、それぞれのテストケースを参照してください。

- [RxMap](https://github.com/sskyy/data0/blob/main/__tests__/rxMap.spec.ts)
- [RxSet](https://github.com/sskyy/data0/blob/main/__tests__/rxSet.spec.ts)
