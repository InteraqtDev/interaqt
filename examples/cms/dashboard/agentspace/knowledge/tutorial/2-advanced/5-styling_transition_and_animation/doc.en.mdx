Axii provides a powerful and flexible styling system that supports various CSS features, including transitions and animations.

## Basic Styling

Axii's styling system supports multiple CSS expression formats:

```jsx
const style = {
    // Standard format
    margin: 10,
    // Array format
    padding: [10, 20],
    // Nesting and pseudo-classes
    '&:hover': {
        background: 'blue'
    },
    // Nested selectors
    '& span': {
        color: 'white'
    }
}
```

## Keyframe Animations

Axii supports defining keyframe animations directly in components:

```jsx
const style = {
    // Define keyframes
    '@keyframes': {
        from: { transform: 'rotate(0deg)' },
        to: { transform: 'rotate(359deg)' }
    },
    // Use animation, @self references the current element's keyframes
    animation: `@self 4s linear infinite`,
    transformOrigin: 'center center'
}
```

## Transition Effects

Axii supports CSS transition effects, which can be defined using array format:

```jsx
const style = {
    transition: ['all 0.3s ease', 'opacity 0.5s ease'],
    '&:hover': {
        opacity: 0.8,
        transform: 'scale(1.1)'
    }
}
```

Axii's styling system is designed to be concise yet powerful, capable of meeting most complex styling needs while maintaining code readability and maintainability.
